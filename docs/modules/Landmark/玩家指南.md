# 地标系统 - 玩家指南

## 什么是地标系统？

地标系统是一个**快速旅行网络**，让你可以在服务器的各个重要地点之间快速传送。

想象一下：你在主城，想去矿区挖矿。以前你可能需要走很远的路，现在只需要打开地标菜单，点一下就能瞬间到达！

## 核心概念

### 解锁地标

- 当你**第一次走进**一个地标区域时，会自动解锁该地标
- 解锁后，你就可以随时传送到这个地标了
- 屏幕上会显示大标题和音效提示你已解锁新地标

### 传送规则

- 你必须**站在任意一个地标区域内**才能发起传送
- 你只能传送到**已解锁**的地标
- 部分地标可能是"默认解锁"的，所有人都能直接传送

## 如何使用

### 打开地标菜单

```
/lm
```

输入这个命令会打开一个图形界面，显示所有地标：

- **绿色图标** = 已解锁，可以传送
- **红色屏障** = 未解锁，需要先去那里探索一次

### 传送到地标

**方法一：通过菜单**
1. 输入 `/lm` 打开菜单
2. 点击你想去的地标（必须是已解锁的）
3. 完成传送！

**方法二：通过命令**
```
/lm tp <地标名称>
```

### 查看地标列表

```
/lm list
```

在聊天框显示所有地标的列表。

### 查看地标详情

```
/lm info <地标名称>
```

查看某个地标的详细信息，包括坐标、是否已解锁等。

## 常见问题

### Q: 为什么我无法传送？

可能的原因：
1. **你不在地标区域内** - 你必须站在任意一个地标里才能传送
2. **目标地标未解锁** - 你需要先亲自走到那个地标一次
3. **传送冷却中** - 刚传送过，需要等待冷却

### Q: 我怎么知道自己在地标里？

当你走进地标区域时，屏幕上会显示：
- 大标题显示地标名称
- 聊天框提示"你已到达: xxx"
- 有音效反馈

### Q: 地标在哪里？

地标通常设置在服务器的重要地点，比如：
- 主城、出生点
- 矿区、农场
- 商店、交易区
- 副本入口
- 各种公共设施

具体位置请咨询服务器管理员或查看服务器公告。

## 小贴士

1. **多探索！** 第一次到达新地方时记得四处走走，说不定就解锁了新地标
2. **记住地标名** 用命令传送时需要输入地标名称
3. **善用菜单** 如果记不住名字，直接 `/lm` 打开菜单更方便
