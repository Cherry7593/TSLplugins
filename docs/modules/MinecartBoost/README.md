# 矿车加速系统

在特定方块上铺设铁轨，可以让载人矿车获得更高的速度。

## 使用方法

1. 在指定方块上方放置任意类型的铁轨
2. 坐上矿车，经过该铁轨时会自动加速
3. 下车后矿车恢复正常速度

## 加速方块一览

| 方块 | 速度 | 倍率 |
|------|------|------|
| 浮冰 (Packed Ice) | 16 m/s | 2× |
| 煤炭块 (Coal Block) | 16 m/s | 2× |
| 雕纹深板岩 (Chiseled Deepslate) | 24 m/s | 3× |
| 磨制玄武岩 (Polished Basalt) | 24 m/s | 3× |
| 切制铜块 (Cut Copper) | 24 m/s | 3× |
| 去皮苍白橡木 (Stripped Pale Oak Wood) | 30 m/s | 3.75× |
| 暗海晶石 (Dark Prismarine) | 30 m/s | 3.75× |
| 雕纹凝灰岩砖 (Chiseled Tuff Bricks) | 30 m/s | 3.75× |

> 原版矿车速度约为 8 m/s

## 注意事项

- 只有**载人**的矿车才会加速，空矿车不受影响
- 支持所有类型的铁轨（普通铁轨、动力铁轨、探测铁轨、激活铁轨）
- 玩家下车后，矿车速度会自动恢复为原版速度
