# 小镇 PHome 系统 - 玩家指南

## 简介

小镇 PHome 是一个**共享传送点系统**，让同一小镇的成员可以共享传送点。与个人家不同，PHome 属于整个小镇，所有小镇成员都可以使用。

---

## 基本命令

| 命令 | 说明 | 谁可以使用 |
|------|------|-----------|
| `/tsl phome` | 查看小镇的 PHome 列表 | 所有成员 |
| `/tsl phome list` | 查看小镇的 PHome 列表 | 所有成员 |
| `/tsl phome <名称>` | 传送到指定 PHome | 所有成员 |
| `/tsl phome tp <名称>` | 传送到指定 PHome | 所有成员 |
| `/tsl phome set <名称>` | 在当前位置创建 PHome | 镇长/副镇长 |
| `/tsl phome del <名称>` | 删除指定 PHome | 镇长/副镇长 |
| `/tsl phome help` | 查看帮助信息 | 所有成员 |

---

## 权限说明

### 镇长 / 副镇长
- ✅ 创建新的 PHome
- ✅ 删除已有的 PHome
- ✅ 覆盖（更新）已有的 PHome
- ✅ 传送到任意 PHome
- ✅ 查看 PHome 列表

### 普通成员
- ❌ 不能创建 PHome
- ❌ 不能删除 PHome
- ✅ 传送到任意 PHome
- ✅ 查看 PHome 列表

### 无小镇玩家
- ❌ 无法使用任何 PHome 功能
- 💡 请先加入一个小镇

---

## 数量限制

每个小镇的 PHome 数量上限取决于**小镇等级**：

| 小镇等级 | PHome 上限 |
|---------|-----------|
| 1 级 | 3 个 |
| 2 级 | 5 个 |
| 3 级 | 8 个 |
| 4 级及以上 | 10 个 |

> ⚠️ **注意**：
> - 达到上限后，无法创建新的 PHome，但可以覆盖已有的
> - 小镇升级后，可立即创建更多 PHome
> - 小镇降级后，已有的 PHome 不会被删除，只是无法新增

## 使用示例

### 查看小镇的 PHome
```
/tsl phome
```
输出示例：
```
=== 千年科技 的PHome列表 (2/5) ===
- 矿场 [world]
- 农场 [world]
```

### 传送到 PHome
```
/tsl phome 矿场
```
或
```
/tsl phome tp 矿场
```

### 创建新 PHome（仅限镇长/副镇长）
站在想要设置的位置，输入：
```
/tsl phome set 仓库
```

### 删除 PHome（仅限镇长/副镇长）
```
/tsl phome del 仓库
```

### 更新已有 PHome 的位置
如果要移动某个 PHome 的位置，只需在新位置再次使用 `set` 命令：
```
/tsl phome set 矿场
```
这会将「矿场」的传送点更新为当前位置。

## 常见问题

### Q: 为什么我无法使用 PHome 命令？
**A:** 可能的原因：
1. 你还没有加入任何小镇
2. 小镇 PHome 功能未在服务器启用

### Q: 为什么我无法创建 PHome？
**A:** 可能的原因：
1. 你不是镇长或副镇长
2. 小镇已达到 PHome 数量上限

### Q: 我可以传送到其他小镇的 PHome 吗？
**A:** 不可以。PHome 是小镇专属的，只有同一小镇的成员才能使用。

### Q: PHome 名称可以包含空格吗？
**A:** 不建议使用空格。推荐使用简短的中文或英文名称，如「矿场」「农场」「home」等。
