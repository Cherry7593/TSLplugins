# Kiss 功能 - 快速使用指南

## 🎯 功能介绍
玩家互动亲吻功能，带有爱心粒子效果和音效。

**所有玩家默认可用，无需权限！**

---

## 📝 使用方法

### 玩家命令
```
/tsl kiss <玩家名>    亲吻指定玩家
/tsl kiss toggle      开启/关闭亲吻功能
```

### 快捷操作
```
Shift + 右键玩家 → 亲吻该玩家
```

---

## 🔧 个人开关说明

### `/tsl kiss toggle` 的作用

- **开启状态**（默认）：
  - ✅ 可以亲吻别人
  - ✅ 可以被别人亲吻

- **关闭状态**：
  - ❌ **不能**亲吻别人（静默拒绝）
  - ✅ **仍可以**被别人亲吻

> **注意**：个人开关只控制你能否亲吻别人，不影响别人亲吻你。

---

## ⚙️ 配置

### config.yml
```yaml
kiss:
  enabled: true        # 是否启用
  cooldown: 1.0       # 冷却时间（秒）
```

## ⏱️ 冷却机制

- 默认冷却时间：1秒
- 冷却中**静默拒绝**（无提示）
- OP 和拥有 bypass 权限的玩家无视冷却

## 💡 使用场景

### 场景 1：我不想主动亲吻别人
```
执行：/tsl kiss toggle（关闭功能）
效果：你不能亲吻别人，但别人仍可以亲吻你
```

### 场景 2：我想恢复正常使用
```
执行：/tsl kiss toggle（开启功能）
效果：你可以亲吻别人，也可以被别人亲吻
```

## ❓ 常见问题

**Q: 我关闭了功能，为什么还是会被别人亲吻？**  
A: 个人开关只控制你能否主动亲吻别人，不影响别人亲吻你。

**Q: 为什么我点击后没有反应？**  
A: 可能是以下原因：
- 你关闭了亲吻功能（执行 toggle 开启）
- 你在冷却时间内（静默拒绝）
- 目标玩家不在线

**Q: 如何完全避免被亲吻？**  
A: 目前无法拒绝被亲吻，这是为了增加玩家互动的趣味性。
